## About

**STANKIN Schedule** is an unofficial mobile application for MSTU "STANKIN" students that provides convenient access to class schedules, module journal, and university news directly from your smartphone.

This project is a fork of [ProjectPepega](https://github.com/Nikololoshka/ProjectPepega) ‚Äì the original application that is no longer maintained by the author. The project has been revived and continues to evolve with improvements and new features.

### Key Features

- **Class Schedule** ‚Äì view schedules with convenient day navigation
- **Module Journal** ‚Äì access to grades and ratings
- **University News** ‚Äì fresh news directly in the app
- **Home Screen Widget** ‚Äì quick access to schedule without opening the app
- **Dark Theme** ‚Äì comfortable use at any time of day
- **Schedule Import** ‚Äì support for importing from various sources, including MSTU "STANKIN" Moodle

## Technologies

The application is built on modern Android development technologies:

- **Kotlin 2.3.0** ‚Äì modern programming language
- **Jetpack Compose** ‚Äì declarative UI framework
- **Material Design 3** ‚Äì modern design language
- **Hilt** ‚Äì dependency injection
- **Room** ‚Äì local database
- **Retrofit** ‚Äì network requests
- **Firebase** ‚Äì analytics and crash reporting
- **Coroutines** ‚Äì asynchronous programming

## Development

### Requirements

- Android Studio Otter or newer
- JDK 21
- Android SDK 36
- Minimum Android version: 8.0 (API 26)

### Building the Project

```bash
# Clone the repository
git clone https://github.com/overklassniy/STANKIN_Schedule_app.git
cd STANKIN_Schedule_app

# Build the project
./gradlew assembleDebug

# Or open in Android Studio
```
